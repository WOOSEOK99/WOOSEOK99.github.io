🚀 Service Update Notice

Version 1.0.21 Update Details

  • 1. Fixed an issue where some games would crash unexpectedly.
  • 2. Implemented a new D-pad control scheme. (This addresses the issue where simultaneous key presses were not recognized on certain devices.)
  • 3. Removed the D-PAD / Analog selection in Settings, as it is no longer required with the new control scheme.

Version 1.0.20 Update Details

  • 1. Online Help Added - Click the "!" icon next to the top bar settings to access help, user manuals, and update information.
  • 2. Fixed an issue causing abnormal termination on some phones.
  • 3. Fixed an issue where the search function in the Home-All Games list was not working.
